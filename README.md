Contributing
-------
１．システムを動かしてみて、不具合を見つけたり、追加したい機能が浮かんだら、イシューを追加してください:hand:  
２．イシューの中から自分ができそうなものを選んで、自PCにクローンしたリポジトリに実装してみてください:computer:   
３．実装してテストをしたら、プルリクエストを送ってください。レビューしてmasterブランチにマージします:thumbsup:
